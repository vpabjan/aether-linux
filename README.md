# Aether Linux

Aether Linux is an arch-based distro aiming for user-friendliness, with all of the cool stuff.\
Preinstalled stuff:
 - Hyprland (Desktop environment)
 - Alacritty (Terminal)
 - Rofi (drun)
 - Firefox (browser)
 - Nemo (File manager)

Helper utilities for getting drivers, software, changing settings and other features are being worked on.
